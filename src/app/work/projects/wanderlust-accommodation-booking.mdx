## Overview

A comprehensive full-stack accommodation booking web application inspired by modern travel and rental platforms. The platform enables users to browse listings, search accommodations, upload images, and interact with location-based features.

## Key Features

- **MVC Architecture**: Implemented a scalable and maintainable Model-View-Controller code structure for clean separation of concerns.
- **RESTful APIs**: Developed robust API endpoints using Express.js for all CRUD operations.
- **Cloud Database**: Integrated MongoDB Atlas for cloud-based scalable database management.
- **Image Upload**: Added image upload functionality using Cloudinary for seamless media management.
- **Interactive Maps**: Integrated MapTiler for interactive location-based mapping and accommodation discovery.
- **Responsive UI**: Built a fully responsive user interface using HTML, CSS, and EJS templating.
- **Input Validation**: Implemented comprehensive input validation and error handling mechanisms.

## Technologies Used

- **Node.js & Express.js**: Backend server framework with RESTful API architecture.
- **MongoDB Atlas**: Cloud-hosted NoSQL database for scalable data management.
- **Cloudinary**: Cloud-based image upload and management service.
- **MapTiler**: Interactive mapping and geolocation services.
- **EJS**: Server-side templating for dynamic HTML rendering.
- **HTML & CSS**: Responsive frontend design and layout.

- Full-stack MERN development capabilities

- Cloud service integration (database + media)

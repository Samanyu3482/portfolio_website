## Overview

meTTaFlights is an intelligent flight route optimization platform that combines graph algorithms with AI-driven logic to find the most efficient travel routes. The platform won **1st Runner-Up at HackIndia National Finals 2025** and **1st Position at BGI International Hackathon 2025**.

## Key Features

- **AI-Powered Route Optimization**: Leverages graph algorithms and AI-driven logic to calculate the most efficient flight routes based on multiple parameters including cost, duration, and layover preferences.
- **Intelligent Flight Search**: Built an advanced search system with smart recommendations powered by machine learning models.
- **Automated Booking Integration**: Developed automation workflows for seamless flight booking with intelligent travel assistance through meTTa AI.
- **Real-Time Data Processing**: Processes real-time flight data to provide accurate pricing and availability information.

## Technologies Used

- **Node.js & Express.js**: Backend server architecture and REST API development.
- **React.js**: Dynamic and responsive frontend user interface.
- **MongoDB**: NoSQL database for managing flight data and user preferences.
- **Python**: AI/ML algorithms for route optimization and recommendation systems.
- **Graph Algorithms**: Dijkstra's and A* algorithms for optimal pathfinding.

## Achievements

- 🥇 **1st Position** — BGI International Hackathon 2025
- 🥈 **1st Runner-Up** — HackIndia National Finals 2025

## Impact

The platform demonstrates how AI can revolutionize travel planning by combining graph theory with machine learning to deliver optimized, cost-effective travel solutions. The multi-hackathon success validates the technical approach and real-world applicability of the solution.

## Overview

Urban Mind is a sustainability-focused web platform designed to help users track and understand their carbon emissions. The project secured **2nd Position at the AI for Sustainability Hackathon**.

## Key Features

- **Carbon Footprint Tracking**: Comprehensive tools for users to monitor and track their carbon emissions across daily activities.
- **Environmental Impact Monitoring**: Real-time environmental data visualization with actionable sustainability insights.
- **Data Visualization**: Interactive charts and graphs that make environmental impact data easy to understand.
- **AI-Powered Insights**: Machine learning models to analyze patterns and suggest emission reduction strategies.
- **Sustainability Awareness**: Educational components to increase awareness about environmental impact.

## Technologies Used

- **React.js**: Dynamic and responsive frontend interface.
- **Node.js & Express.js**: Backend API for data processing and analytics.
- **MongoDB**: Database for storing user emission data and environmental metrics.
- **Python**: AI/ML models for emission pattern analysis.
- **Chart.js**: Interactive data visualization for environmental metrics.

## Achievements

- 🥈 **2nd Position** — AI for Sustainability Hackathon

## Impact

Urban Mind empowers individuals to understand and reduce their environmental impact through data-driven insights and AI-powered recommendations, contributing to a more sustainable future.

## Overview

ArthSaarthi is a user-friendly financial analysis platform designed to simplify stock and mutual fund tracking for everyday investors. The platform secured **2nd Position at Build for Bharat National Hackathon**.

## Key Features

- **Stock Analysis Dashboard**: Real-time stock tracking with comprehensive analytical tools and data visualizations.
- **Mutual Fund Tracking**: Simplified mutual fund monitoring with performance metrics and comparison tools.
- **User-Friendly Insights**: Focused on making financial data accessible and understandable for users with varying levels of financial literacy.
- **Analytical Dashboards**: Interactive dashboards with charts, graphs, and key financial indicators.
- **Investment Monitoring**: Portfolio tracking with automated alerts and performance summaries.

## Technologies Used

- **React.js**: Component-based frontend with dynamic data rendering.
- **Node.js & Express.js**: RESTful API backend for data processing.
- **MongoDB**: NoSQL database for user portfolios and financial data storage.
- **JavaScript**: Core logic for data visualization and interactive charts.

## Achievements

- 🥈 **2nd Position** — Build for Bharat National Hackathon

## Impact

ArthSaarthi bridges the gap between complex financial data and everyday investors by providing intuitive, accessible tools for investment tracking and analysis.

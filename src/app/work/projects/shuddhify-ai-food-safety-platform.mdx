## Overview

Shuddhify.AI is an AI-powered platform focused on food adulteration detection and food safety awareness. The platform combines AI-based image analysis, community reporting, and automation workflows to improve public food safety accessibility.

## Key Features

- **AI Image Analysis**: Developed image-based adulteration detection using AI classifiers powered by Gemini API for real-time food quality assessment.
- **AI Voice Agent**: Integrated AI voice-agent support for lab booking and food-testing queries, making the platform accessible to all users.
- **Community Reporting**: Built a community-driven reporting system with automated location capture using Leaflet maps integration.
- **Educational Modules**: Added comprehensive modules about common food adulterants and safety practices to spread awareness.
- **Automation Workflows**: Implemented n8n-based workflow automation for notifications, reporting, and data processing.
- **Secure Authentication**: Integrated Auth0 for robust user authentication and authorization.

## Technologies Used

- **Node.js & Express.js**: Backend API development and server-side logic.
- **MongoDB**: NoSQL database for storing reports, user data, and educational content.
- **Gemini API**: AI-powered image classification and natural language processing.
- **n8n**: Workflow automation for notifications and data processing pipelines.
- **Leaflet**: Interactive maps for location-based community reporting.
- **Gmail API**: Automated email notifications and alerts.
- **Auth0**: Secure authentication and user management.
- **JavaScript**: Frontend logic and interactive UI components.
